# 行動規範

> **English** — [English version here](../CODE_OF_CONDUCT.md)

## 私たちの誓い

メンバー・コントリビューター・メンテナーとして、年齢・体型・障害・民族・性自認と表現・経験レベル・国籍・外見・人種・宗教・性的指向に関わらず、すべての人にとってハラスメントのない体験を提供することを誓います。

## 私たちの基準

良好な環境づくりに貢献する行動の例:

- 歓迎的で包括的な言葉を使う
- 異なる視点や経験を尊重する
- 建設的な批判を誠実に受け入れる
- コミュニティにとって最善のことに集中する
- 他のコミュニティメンバーへの共感を示す

受け入れられない行動の例:

- 性的な言葉や画像の使用、不適切な性的注目やアドバンス
- トローリング、侮辱的・中傷的なコメント、個人的または政治的攻撃
- 公的または私的な嫌がらせ
- 明示的な許可なく他者の個人情報を公開すること
- 職業的な場で不適切とみなされるその他の行為

## メンテナーの責任

プロジェクトメンテナーは、許容される行動の基準を明確にし、不適切な行動に対して公正な是正措置を取る責任があります。

メンテナーは、この行動規範に沿わないコメント・コミット・コード・Issue・その他の貢献を削除・編集・拒否する権利および責任を持ちます。

## 適用範囲

この行動規範は、GitHub リポジトリ・Issue・Pull Request・その他プロジェクトが使用するすべてのコミュニケーションチャンネルに適用されます。

## 報告

虐待的・嫌がらせ・不適切な行動を発見した場合は、Issue を作成するか GitHub を通じてメンテナーに直接連絡してください。すべての報告は迅速かつ公正に審査されます。

## 帰属

この行動規範は [Contributor Covenant](https://www.contributor-covenant.org) バージョン 2.1 を基に作成されています。
